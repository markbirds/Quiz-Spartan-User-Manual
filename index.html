What is Quiz Spartan?
Quiz Spartan is a web-based assessment tool developed for Batangas State University. This tool can be used to create traditional assessments which includes identification, multiple choice and essay type of questions. It can also be used to create programming quizzes with supported programming languages including C++, Java and Python. Since assessments created using Quiz Spartan will be answered online, the system can detect academic dishonesties which includes switching of tabs, opening another application and similarity in essay answers.

Logging in to Quiz Spartan
Since Quiz Spartan was developed for Batangas State University, users are expected to login their G-Suite accounts registered on the university. Users can login in this URL https://quiz-spartan.herokuapp.com/login/.

To create different files in Quiz Spartan, use the create button located on the top-right side of the tab content.

By using the search bar on the top of the page, users can search files based on the filename or data it was modified.

Adding question

Users can add question by choosing the question type on the sidebar and clicking the add question button. The newly created question will be placed after the currently focused question.

Question types

Currently, the supported question types in Quiz Spartan are identification, multiple choice, essay ,checkbox, dropdown and file upload. The sections below further discusses each of the question types.

  • Identification - An identification type of question is where the examinee are expected to clearly identify or define each item. To add correct answer, click the "Add correct answer" input. To remove correct answer, click the cross button beside the correct answer. You can also set the case sensitivity for the answers.
  • Multiple Choice - It is a form of an objective assessment in which respondents are asked to select only correct answers from the choices offered as a list. To add choices, click the Add option component. You can either click the radio button or the switch to set the option as the correct answer. Options are easily removed just by clicking the cross button beside the correct answer switch. Note that multiple choice question only accepts one correct answer. If the you want to have multiple correct answers, you can use checkbox.
  • Essay - Since essays are subjective type of questions, setting correct answers are not needed.
  • Checkbox - This type of question is similar to multiple choice except multiple correct answers can be set. Setting correct answers can be done by clicking the checkbox or the correct answer switch. Removing options is similar to multiple choice.
  • Dropdown - This question is very similar to multiple choice and it also accepts only one correct answer. However, it is quite useful to use if options are too many.
  • File upload - This allows the examinees to upload their files. Assessment owner can set specific type of file allowed such as word document, spreadsheet, PDF, presentation, video or image.
Adding section

Sections are great in separating the context pf questions. You can add section by clicking the dropdown inside the add question button.

Other functionalities

This section discusses the functionalities of the components at the bottom of the question in sequence from left to right.

  • Adding points - Located at the left side of the bottom component, users can enter the points for the focused question.
  • Move up - This moves the current focused question up.
  • Move down - This moves the current focused question down.
  • Duplicate - Duplicates the current focused question.
  • Add youtube video - Adds a youtube video to the question. To add a video, get the ID from youtube video URL. Youtube id will be found in https://www.youtube.com/watch?v=id
  • Add image - Adds an image to the quesiton. Images should be uploaded to Images folder inside Quiz Spartan folder on your Google Drive.
  • Delete - Deletes the current focused question.

Adding programming problem

Users can add a programming problem clicking the add button. The language for each problem will based on the chosen language in the sidebar.

Problem statement and test cases

The first input on the top of the problem is where you would put the problem statement. In the sample image above, the problem statement is "Write a program in Python to add two numbers". A test case is a way to test and score the submitted source code. There are two types of test case in Quiz Spartan: visible test case and hidden test case. Unlike hidden test cases, visible test cases are not scored since examinees can see the input and expected output, they can easily satisfy the expected output through printing. Setting multiple inputs can be done by entering the subsequent inputs on the next line.

Settings
Users can set the settings in created assessments and programming quizzes by clicking the settings button located on the top of the page. The settings is divided into three sections: time, general and assessment.
  • Time - Assessment owners can set whether to accept or not accept submissions. If enabled, examiners can enter the start and end time of the assessment. If left blank, the assessment can be accessed anytime.
  • General - In general settings, examiners can enable retaking of assessment, shuffling of question order and releasing scores after submission.
  • Assessment - This contains configurations for detecting switching of tabs and displaying the essay similarity table. The latter will only be available in assessments not in programming quizzes.
Submissions
Submissions tab holds all the information concerning the submitted assessments. It includes summary of submissions, individual submissions and functionalities such as releasing all scores, saving emails addresses to create group email and exporting the summary of submissions to Google Sheets.
Summary of submissions

The summary of submissions contains set of tables containing information for all submissions and detected academic dishonesties. The all submissions table displays the name, email, time took to finished answering, score, and attempt number of the examinee. The tables for detected academic dishonesties are the switched tabs table and the essay similarity table which are both displayed below the all submissions table.
